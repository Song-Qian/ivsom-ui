功能按钮

Icon 属性接口

  • 方式一：功能性按钮一般是支持ICON属性，向用户表达该按钮的功能用途
  • 方式二：框架里面的ICON是固定有限的，在特定的业务中，功能性按钮需要通过插槽功能加入自定义ICON，向用户表达不同意义的按钮的功能用途

按钮事件

Event 接口

  • onClick: Arguments(MouseEvent) 按钮点击事件
 打印按钮  确认按钮  查询按钮  警示按钮  删除按钮 
<i-vsom-button @click="onBtnClick.apply(this, ['点击了普通按钮'])"><i class="iconfont">&#xe6e4;</i> 打印按钮</i-vsom-button>
 <i-vsom-button @click="onBtnClick.apply(this, ['点击了确认按钮'])" type="success"><i class="iconfont">&#xe6d5;</i> 确认按钮</i-vsom-button>
 <i-vsom-button @click="onBtnClick.apply(this, ['点击了查询按钮'])" type="primary"><i class="iconfont">&#xe6d6;</i> 查询按钮</i-vsom-button>
 <i-vsom-button @click="onBtnClick.apply(this, ['点击了警示按钮'])" type="warning"><i class="iconfont">&#xe705;</i> 警示按钮</i-vsom-button>
 <i-vsom-button @click="onBtnClick.apply(this, ['点击了删除按钮'])" type="danger"><i class="iconfont">&#xe6f9;</i> 删除按钮</i-vsom-button>
