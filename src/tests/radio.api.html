基础用法

常规单选框

  • Radio 组件在使用之前，需要有相关的依赖库进行支持，如需阅读：快速开始
男 女 
<i-vsom-radio v-model="radio_value" value="男">男</i-vsom-radio>
 <i-vsom-radio v-model="radio_value" value="女">女</i-vsom-radio>

不同大小的单选框

  • Radio size属性支持 medium、small、mini
medium small mini 
<i-vsom-radio size="medium" v-model="radio_value" value="medium">medium</i-vsom-radio>
 <i-vsom-radio size="small" v-model="radio_value" value="small">small</i-vsom-radio>
 <i-vsom-radio size="mini" v-model="radio_value" value="mini">mini</i-vsom-radio>

不同类型的单选框

  • Radio type属性支持 circle、check
男 女 男 女 
<i-vsom-radio size="small" v-model="radio_value1" value="男">男</i-vsom-radio>
<i-vsom-radio size="small" v-model="radio_value1" value="女">女</i-vsom-radio>
<i-vsom-radio type="check" size="small" v-model="radio_value2" value="男">男</i-vsom-radio>
<i-vsom-radio type="check" size="small" v-model="radio_value2" value="女">女</i-vsom-radio>

被禁用的单选框

禁用样式一 禁用样式二 
<i-vsom-radio size="small" checked="1" value="1" disabled>禁用样式一</i-vsom-radio>
 <i-vsom-radio type="check" size="small" checked="2" value="2" disabled>禁用样式二</i-vsom-radio>

一组单选框

  • RadioGroup name属性是Radio按钮组的表单参数名，在form表单提交时非常有用!
  • RadioGroup value属性是Radio选中的值
男 女 
<i-vsom-radio-group v-model="radio_value3" name="sex">
    <i-vsom-radio type="check" :checked="radio_value3" size="small" value="男">男</i-vsom-radio>
    <i-vsom-radio type="check" :checked="radio_value3" size="small" value="女">女</i-vsom-radio>
 </i-vsom-radio-group>

RadioGroup Event 事件

常规单选框

  • onChange: Arguments(value : string) Radio 引发值改变的事件。
男 女 
<i-vsom-radio-group v-model="radio_value" name="sex" @on-change="onChange">
    <i-vsom-radio type="check" :checked="radio_value" size="small" value="男">男</i-vsom-radio>
    <i-vsom-radio type="check" :checked="radio_value" size="small" value="女">女</i-vsom-radio>
 </i-vsom-radio-group>
