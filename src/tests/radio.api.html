Radio Api

  • CDN
  • NodeJS

基础用法

常规单选框

  • Radio 组件在使用之前，需要有相关的依赖库进行支持，如需阅读：快速开始
<i-vsom-radio :options="testArray" :selectOption="1"  @radio-change="setRadioValue"></i-vsom-radio>   
                            

                            var app = new Vue({
                                el : '#app',
                                data : function() {
                                    return {
                                        testArray:[{
                                            name:"苹果",
                                            value:1,
                                           
                                        },{
                                            name:"香蕉",
                                            value:2,
                                            
                                        },{
                                            name:"梨子",
                                            value:3,
                                            
                                        }],
                                        select:1,
                                    }
                                },
                                methods : {
                                    setRadioValue:function (value) {
                                        const { val } = value.target 
                                      console.log(value.target.value);
                        
                                    }
                                }
                            })
