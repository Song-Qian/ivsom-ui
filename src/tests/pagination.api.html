基础用法

常规分页用法

  • pagination 组件在使用之前，需要有相关的依赖库进行支持，如需阅读：快速开始 
<i-vsom-pagination :currentPage.sync="cur" :totalcount.sync="all" v-on:change-handle="listen" ></i-vsom-pagination>   
                            

                            var app = new Vue({
                                el : '#app',
                                data : function() {
                                    return {
                                        cur: 1,//当前页数
                                        all: 500,//总条数
                                        limit:10//每页显示条数
                                    }
                                },
                                listen: function (data) {
                                    console.log(data)
                                }
                               
                            })
