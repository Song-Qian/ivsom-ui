基础用法

默认样式

  • Table 组件在使用之前，需要有相关的依赖库进行支持，如需阅读：快速开始
  • Table 组件具有默认的宽度 (100%) 和高度 (auto)
  • iVsomTable 与 iVsomTableColumn 需配合使用，单独使用任一组件，将会无法达到预定期望。
  • Table Props:
  • width: number | string Table 宽度
  • height: number | string Table 高度
  • row-style: object 表格的行样式
  • col-style: object 表格的单元格样式
  • border: boolean 是否显示边框
  • stripe: boolean 是否显示斑马纹
  • round: boolean 是否圆角
  • highlight-row: boolean 是否高亮行
  • source: Array 数据源
  • empty-text: string 空内容显示
  • Table Column Props:
  • width: number | string 列的宽度
  • Flex: boolean 是否固定列
  • FlexAlign: "start" | "end" 固定列位置
  • HeaderAlign: 'left' | 'center' | 'right' 头部标题对齐方式
  • Align: 'left' | 'center' | 'right' 列内容对齐方式
  • title: string 列标题
  • Field: string 列字段
  • Sort: boolean 是否启用排序
  • Table Event:
  • onCellClick: (e: MouseEvent, data : any, rowIndex: number, field: string) Table单元格点击事件
  • onCellDbclick: (e: MouseEvent, data : any, rowIndex: number, field: string) Table单元格双击事件
  • onRowHighlight: (e: MouseEvent, data : any, rowIndex: number, field: string) Table表格行鼠标移入行高亮事件
  • onTableSort: (e: MouseEvent, data : any, rowIndex: number, field: string) Table表格排序事件

var app = new Vue({
    el : '#app',
    methods : {
        handlerRowStyle: function(index, data) {
            if(index % 4 === 0) {
                return { 'background-color' : '#2999f7' }
            }

            if(index % 4 === 1) {
                return { 'background-color' : '#3bd134' }
            }

            if(index % 4 === 2) {
                return { 'background-color' : '#ecc025' }
            }

            return { 'background-color' : '#f75f3c' }
        },
        handlerColStyle: function(rowIndex, value) {
            if( value > 50) {
                return { color : 'red' }
            }
            return { color : '#fff' }
        }
    }
 })

var app = new Vue({
    el : '#app',
    methods : {
        onCellClick: function(e, row, rowIndex, field) {
            var me = this;
            me.$ivsom_message.prompt({ title : '提示', type : 'success', message : '当前点击的是第' + rowIndex + '行,' + field + '字段!' });
        },
        onCellDbClick: function(e, row, rowIndex, field) {
            var me = this;
            me.$ivsom_message.prompt({ title : '提示', type : 'success', message : '当前双击的是第' + rowIndex + '行,' + field + '字段!' });
        },
        onRowHighlight: function(rowIndex) {
            var me = this;
            me.$ivsom_message.prompt({ title : '提示', type : 'success', message : '当前高亮行是' + rowIndex + '行' });
        },
        onTableSort: function(field, sort) {
            var me = this;
            var isAsc = sort === 'asc' ? '正序' : '倒序';
            me.$ivsom_message.prompt({ title : '提示', type : 'success', message : '当前'+ isAsc +'排序字段是' + field });
        }
    }
 })
