基础用法

普通菜单

  • Message 组件在使用之前，需要有相关的依赖库进行支持，如需阅读：快速开始
  • Message 公共属性:
  • message: string 消息内容
  • mask: boolean 是否遮罩
  • target: VNode 消息框弹出时，追加到哪个元素里面。
  • isDestroy: boolean 消息框移除时，是否销毁。
  • Alert Props :
  • mask: boolean 默认值 false
  • isDestroy: boolean 默认值 true
  • type: "success" | "primary" | "info" | "warning" | "error" 消息框的图标类型
  • title: string 消息框的标题头
  • width: number | string 消息框的宽度
  • height: number | string 消息框的高度
  • isOuterClose: boolean 是否点击外部关闭按钮, mask === true 时生效。
  • Toast Props :
  • mask: boolean 默认值 true
  • isDestroy: boolean 默认值 true
  • message: string Toast此属性无效
  • Prompt Props :
  • mask: boolean Prompt此属性无效
  • isDestroy: boolean 默认值 true
  • timeout: number 消息停留时间
  • type: number 消息类型

消息框的关闭事件

alert 消息框 
<i-vsom-button type="primary" icon="icon-gongnengtubiao-60" @click="openAlertOn({ title : '提示', message : '这是一个普通的消息框', mask: true, target : $root })">alert 消息框</i-vsom-button>

var app = new Vue({
    el : '#app',
    data : function() {
        return {
        }
    },
    methods : {
        openAlertOn: function(opts) {
            var me = this;
            me.$ivsom_message.alert(opts, function(e){
                alert("消息框被关闭了");
            });
        }
    }
})

确认框的事件

确认框 
<i-vsom-button type="primary" icon="icon-gongnengtubiao-60" @click="openConfirmOn({ title : '提示', message : '这是一个普通的确认框', isDestroy : true, target: $root })">确认框</i-vsom-button>

var app = new Vue({
    el : '#app',
    data : function() {
        return {
        }
    },
    methods : {
        openConfirmOn: function(opts) {
            var me = this;
            me.$ivsom_message.confirm(opts, function(e) {
                alert("yes")
            }, function(e) {
                alert("no")
            });
        }
    }
})

如何手动关闭状态框

状态框 
<i-vsom-button type="primary" icon="icon-gongnengtubiao-60" @click="openToast({ target: $root, mask: true, isDestroy: true })">状态框</i-vsom-button>

var app = new Vue({
    el : '#app',
    data : function() {
        return {
        }
    },
    methods : {
        openToast: function(opts) {
            var me = this;
            var toast = me.$ivsom_message.toast(opts);
            setTimeout(function() {
                toast.Hide();
            }, 3000)
        }
    }
})
