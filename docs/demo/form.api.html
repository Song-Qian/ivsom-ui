表单的验证

  • prop: String 验证字段
  • required： Boolean 是否开启必填
  • min： Number 验证表单属性最小值，表单值是Number类型则验证表单值>= min，表单值是String类型则验证表单值length >= min，推荐：Number类型表单使用v-model.number修饰符
  • max： Number 验证表单属性最小值，表单值是Number类型则验证表单值<= max，表单值是String类型则验证表单值length <= max，推荐：Number类型表单使用v-model.number修饰符
  • e-mail: Boolean 是否开启电子邮箱验证
  • phone: Boolean 是否开启手机、电话坐机验证
  • double: Boolean 是否开启小数精度验证
  • identifier: Boolean 是否开启身份证验证，支持15位一代身份证、18位二代身份证。
  • validate： RegExp | Function 表单的自义定验证
  • error-message: 上述属性，验证失败显示消息内容。

自定义表单验证

提交 清空 
<i-vsom-form ref="form2" :model="Form1" :label-width="80" position="left">
    <i-vsom-form-validater label="密码" prop="password" error-message="请输入密码，长度在6-12位长度。"
        :validate="/^[a-zA-Z0-9]{6,12}$/" required>
        <i-vsom-input v-model="Form1.password" prefix="icon-gongnengtubiao-125"></i-vsom-input>
    </i-vsom-form-validater>
    <i-vsom-form-validater label="确认密码" prop="confirmPassword" error-message="新密码与密码不相等"
        :validate="confirmForm" required>
        <i-vsom-input v-model="Form1.confirmPassword" prefix="icon-gongnengtubiao-125">
        </i-vsom-input>
    </i-vsom-form-validater>
</i-vsom-form>
<i-vsom-button-group>
    <i-vsom-button type="primary" @click="submitForm('form2')">提交</i-vsom-button>
    <i-vsom-button type="danger" @click="clearForm('form2')">清空</i-vsom-button>
</i-vsom-button-group>

var app = new Vue({
    el : '#app',
    data : function() {
        return {
            Form1: {
                password: '',
                confirmPassword: ''
            }
        }
    },
    methods: {
        submitForm: function (form) {
            var me = this;
            me.$refs[form].validateForm().then(function (valid) {
                if (valid) {
                    alert("验证成功");
                }
            })
        },
        clearForm: function (form) {
            var me = this;
            me.$refs[form].clearValidates();
        },
        confirmForm: function (callback) {
            var me = this;
            if (!me.$refs.form2.validateField("password")) {
                return callback(new Error("原密码验证失败"));
            }

            if (me.Form1.password !== me.Form1.confirmPassword) {
                return callback(new Error());
            }
            return callback();
        }
    }
})
