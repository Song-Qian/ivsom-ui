快速开始

CDN 环境引用

CDN 环境是最前端UI最初引入方式，特点是：

  • 直接使用link, scripts 等脚本语言去引入所需要的依赖项
  • 现代浏览器中可以使用defer等技术实现异步加载

依赖环境：

iVsomUI 是在微创光电内部基于现有产品需求，规划的一套UI组件库，当前开发环境是Vue 2.6.11与TSX语言结合进行开发，所以在使用本套UI框架前，必须引用相关的依赖支持库。

第一步：引用iVsomUI.css文件

<!-- 引用全局样式文件 -->
 <link rel="stylesheet" href="iVsomUI.css" />

第二步：引入相关的依赖JS库

<script src="vue.min.js"></script>
 <script src="iVsom.min.js"></script>

第三步：开始代码

Vue.use(iVsomUI);
 var app = new Vue({
    el : '#app',
    data : function() {
        return {
        }
    },
    methods : {
        onClick : function(e) {
            console.log("submit click", e);
        }
    }
 })

注意：我们还继续支持AMD & UMD规范的引用，方便在RequireJS环境下使用
