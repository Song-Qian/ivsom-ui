基础用法

常规使用

  • Select 组件在使用之前，需要有相关的依赖库进行支持，如需阅读：快速开始
  • label : string 选中的显示的文本
  • data-source: Array 选择框数据源
  • multiple: Boolean 是否支持多选
  • placeholder: string 描述提示信息
  • prefix: string 头部 icon
  • suffix: string 尾部 icon
  • width: string | number 选择框的宽度
  • size: 'medium' | 'small' | 'mini' 表单元素大小
  • method: 'hover' | 'click' 选择框展开下拉框的方式
  • disabled: Boolean 是否禁用
<i-vsom-select placeholder="请选择" v-model="value" :data-source="items" label="name"></i-vsom-select>

选择框的表单验证

多选的验证

<i-vsom-form :model="form" :label-width="80" position="left">
    <i-vsom-form-validater label="喜爱" prop="fruit" :validate="validateForm" error-message="当前选中的水果已售空" required>
        <i-vsom-select suffix="icon-gongnengtubiao-182" method="hover" placeholder="请选择" v-model="form.fruit" multiple="true" :data-source="items" label="name"></i-vsom-select>
    </i-vsom-form-validater>
</i-vsom-form>

var app = new Vue({
    el : '#app',
    data : function() {
        return {
            items : [
                { name : '苹果', value : 1 },
                { name : '草莓', value : 2 },
                { name : '香蕉', value : 3 },
                { name : '荔枝', value : 4 },
                { name : '火龙果', value : 5 },
                { name : '百香果', value : 6 }
            ],
            form : {
                fruit : null
            }
        }
    },
    methods : {
        validateForm : function(callback) {
            var me = this;
            var result = me.form.fruit.map(function(el) { return el.name; });
            if(result.indexOf('百香果') > -1) {
                return callback(new Error('百香果暂未进货'))
            }

            if(result.indexOf('香蕉') > -1) {
                return callback(new Error())
            }

            callback();
        }
    }
})

Select Event 事件

  • onChange : Arguments(e: MouseEvent, value: any) 选择框的选中值变化事件
  • onRemove : Arguments(e: MouseEvent, value: any) 多选模式下的选择框，移除选值事件

onChange 事件

<i-vsom-select placeholder="请选择" v-model="value" :data-source="items" @on-change="onChange" label="name" multiple></i-vsom-select>

var app = new Vue({
    el : '#app',
    data : function() {
        return {
            value : null,
            items : [
                { name : '苹果', value : 1 },
                { name : '草莓', value : 2 },
                { name : '香蕉', value : 3 },
                { name : '荔枝', value : 4 },
                { name : '火龙果', value : 5 },
                { name : '百香果', value : 6 }
            ]
        }
    },
    methods : {
        onChange: function(e, item) {
            alert("name: " + item.name + ", value: " + item.value);
        }
    }
})
