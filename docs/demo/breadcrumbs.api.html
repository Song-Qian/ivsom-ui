基础用法

常规使用

  • Breadcrumbs 组件在使用之前，需要有相关的依赖库进行支持，如需阅读：快速开始
<i-vsom-bread-crumbs :debris='items'></i-vsom-bread-crumbs>

var app = new Vue({
    el : '#app',
    data : function() {
        return {
            items : [
                { text : '面包屑一', to : '#1'},
                { text : '面包屑二', to : '#2'},
                { text : '面包屑三', to : '#3'},
                { text : '面包屑四', to : '#4'},
                { text : '面包屑五', to : '#5'},
                { text : '面包屑六', to : '#6'},
                { text : '面包屑七', to : '#7'}
            ]
        }
    }
})

自定义符号

  • Breadcrumbs 组件支持使用ICON 组件当成面包屑中的分隔符号
<i-vsom-bread-crumbs :debris='items' separator='icon-gongnengtubiao-143'></i-vsom-bread-crumbs>

双向响应

  • Breadcrumbs 组件debris 属性支持使用.sync 修饰符来实现双向响应
<i-vsom-bread-crumbs :debris.sync='items_1' ></i-vsom-bread-crumbs>

面包屑路由功能

启用Vue-Route路由功能

  • Breadcrumbs 组件默认URI跳转路由功能是普通的A标签完成
  • Breadcrumbs 组件支持Vue-Route 静态路由功能
<i-vsom-bread-crumbs :debris.sync='items_2' auto-route></i-vsom-bread-crumbs>

链接打开方式

  • Breadcrumbs 组件Target属性默认使用 "_self"，参数定义和用法
  • "_blank" 在新窗口中打开被链接文档。
  • "_self" 默认。在相同的框架中打开被链接文档。
  • "_parent" 在父框架集中打开被链接文档。
  • "_top" 在整个窗口中打开被链接文档。
  • "framename" 在指定的框架中打开被链接文档。
<i-vsom-bread-crumbs :debris='items_3' target="_blank"></i-vsom-bread-crumbs>
 <i-vsom-bread-crumbs :debris='items_3' target="_self"></i-vsom-bread-crumbs>
 <i-vsom-bread-crumbs :debris='items_3' target="_parent"></i-vsom-bread-crumbs>
 <i-vsom-bread-crumbs :debris='items_3' target="_top"></i-vsom-bread-crumbs>

面包屑事件

Event 事件

  • onJumpTo: Arguments(e : MouseEvent, hash_code : string) 面包屑点击跳转
<i-vsom-bread-crumbs ref="breadcrumbs" :debris.sync='items_4' @on-jump-to='JumpTo'><i-vsom-bread-crumbs>


    Vue.use(iVsomUI);
    Vue.use(VueRouter);

    var app = new Vue({
        el : '#app',
        data : function() {
            return {
                items_4 :  [
                    { text : '面包屑一', to : '#1'},
                    { text : '面包屑二', to : '#2'},
                    { text : '面包屑三', to : '#3'},
                    { text : '面包屑四', to : '#4'},
                    { text : '面包屑五', to : '#5'},
                    { text : '面包屑六', to : '#6'},
                    { text : '面包屑七', to : '#7'}
                ]
            }
        },
        methods : {
            JumpTo : function(args) {
                var me = this;
                var key = args[1];
                alert("路由地址：" +  me.$refs.breadcrumbs.map.get(key).to);
            }
        }
    })
