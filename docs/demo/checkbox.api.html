基础用法

常规复选框

  • Checkbox 组件在使用之前，需要有相关的依赖库进行支持，如需阅读：快速开始
<i-vsom-checkbox :data-source="optionData"></i-vsom-checkbox>   
                            

                            var app = new Vue({
                                el : '#app',
                                data : function() {
                                    return {
                                        optionData:[
                                            {label:'苹果',value:'red'},{label:'香蕉'',value:'write'},
                                        ]
                                    }
                                },
                               
                            })

v-model双向绑定复选框

  • v-model支持双向数据绑定，当默认为空则不选中值，默认有值则选中对应的vaule值
<i-vsom-checkbox v-model="model" @change="change" :data-source="optionData"></i-vsom-checkbox>   
                            

                            var app = new Vue({
                                el : '#app',
                                data : function() {
                                    return {
                                        optionData:[
                                            {label:'苹果',value:'red'},{label:'香蕉',value:'write'},
                                        ],
                                        model:"red"
                                    }
                                },
                                methods:{
                                    change(value){
                                       console.log(value)
                                    }
                                 }
                               
                            })

禁用复选框

  • Checkbox 组件支持禁用属性
  • 当某一个复选框是禁用稍微，只需在对应的数据源里加上disabled字段
<i-vsom-checkbox v-model="model1" @change="change" :data-source="optionData"></i-vsom-checkbox>   
                            

                            var app = new Vue({
                                el : '#app',
                                data : function() {
                                    return {
                                        optionData1:[
                                            {label:'苹果',value:'red'},{label:'香蕉',value:'write',disabled:true},
                                        ],
                                        model1:"write"
                                    }
                                },
                                methods:{
                                    change(value){
                                       console.log(value)
                                    }
                                 }
                               
                            })
