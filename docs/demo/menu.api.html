基础用法

普通菜单

  • Menu 组件在使用之前，需要有相关的依赖库进行支持，如需阅读：快速开始
  • Menu 组件默认自适应外部容器高度与宽度。
一级菜单 - {{ n }} 
<div style="height: 400px; width: 400px; display: inline-block; vertical-align: top;">
    <i-vsom-menu>
        <i-vsom-menu-item v-for='n in 3' :key="n" icon="icon-gongnengtubiao-95">
            一级菜单 - {{ '\u007b\u007b n \u007d\u007d' }}
            <template #submenu>
                <i-vsom-menu-item v-for="k in 3" icon="icon-gongnengtubiao-97">
                    二级菜单 - {{ '\u007b\u007b k \u007d\u007d' }}
                    <template #submenu>
                        <i-vsom-menu-item v-for="j in 3" icon="icon-gongnengtubiao-114">三级菜单 - {{ '\u007b\u007b j \u007d\u007d' }}</i-vsom-menu-item>
                    </template>
                </i-vsom-menu-item>
            </template>
        </i-vsom-menu-item>
    </i-vsom-menu>
</div>

肤色

dark

  • 菜单默认是白色皮肤
  • dark 开启菜单使用专业黑色皮肤，该模式适用各种操作环境

可筛选菜单

search

  • Menu 组件search是开启/关闭筛选功能
  • Menu 组件search-text是筛选过程中输入的文本内容
  • Menu 组件过滤筛选功能需要配合filter属性回调函数完成
一级菜单 - {{ n }} 
<i-vsom-menu :filter="handlerFilter" :search-text.sync="searchText" search dark>
    <i-vsom-menu-item v-for='n in 3' :key="n" icon="icon-gongnengtubiao-95">
        一级菜单 - {{ '\u007b\u007b n \u007d\u007d' }}
        <template #submenu>
            <i-vsom-menu-item v-for="k in 3" icon="icon-gongnengtubiao-97">
                二级菜单 - {{ '\u007b\u007b k \u007d\u007d' }}
                <template #submenu>
                    <i-vsom-menu-item v-for="j in 3" icon="icon-gongnengtubiao-114">
                        三级菜单 - {{ '\u007b\u007b j \u007d\u007d' }}
                        <template #submenu>
                            <i-vsom-menu-item v-for="i in 3" icon="icon-gongnengtubiao-114">四级菜单 - {{ '\u007b\u007b i \u007d\u007d' }}</template>
                        </template>
                    </i-vsom-menu-item>
                </template>
            </i-vsom-menu-item>
        </template>
    </i-vsom-menu-item>
 </i-vsom-menu>

横向菜单

Horizontal

  • Menu 横向菜单使用horizontal属于开启/关闭该功能
  • Menu horizontal 菜单依外部容器的宽度自适可，也可以使用style固定菜单宽度。
  • Menu horizontal 菜单的检索输入框默认是46px宽度占位内容， 激活检索输入框会扩张到180px宽度占位内容，会挤压列表宽度。
一级菜单 - {{ n }} 
<div style="width: 100%; vertical-align: top;">
    <i-vsom-menu :filter="handlerFilter" :search-text.sync="searchText" horizontal search dark>
        <i-vsom-menu-item v-for='n in 3' :key="n" icon="icon-gongnengtubiao-95">
            一级菜单 - {{ '\u007b\u007b n \u007d\u007d' }}
            <template #submenu>
                <i-vsom-menu-item v-for="k in 3" icon="icon-gongnengtubiao-97">
                    二级菜单 - {{ '\u007b\u007b k \u007d\u007d' }}
                    <template #submenu>
                        <i-vsom-menu-item v-for="j in 3" icon="icon-gongnengtubiao-114">
                            三级菜单 - {{ '\u007b\u007b j \u007d\u007d' }}
                            <template #submenu>
                                <i-vsom-menu-item v-for="i in 3" icon="icon-gongnengtubiao-114">四级菜单 - {{ '\u007b\u007b i \u007d\u007d' }}</template>
                            </template>
                        </i-vsom-menu-item>
                    </template>
                </i-vsom-menu-item>
            </template>
        </i-vsom-menu-item>
    </i-vsom-menu>
</div>

Menu Event 事件

  • onMenuItemClick: Arguments(e : MouseEvent) 每个菜单项点击事件。
  • onSearchActivation: Arguments(e : MouseEvent) 横向菜单检索输入框激活事件
  • onSearchDeactivated: Arguments(e : MouseEvent) 横向菜单检索输入框失活事件
一级菜单 - {{ n }} 
<div style="width: 100%; vertical-align: top;">
    <i-vsom-menu @on-menu-item-click="handlerMenuItemClick" horizontal dark>
        <i-vsom-menu-item v-for='n in 3' :key="n" icon="icon-gongnengtubiao-95">
            一级菜单 - {{ '\u007b\u007b n \u007d\u007d' }}
