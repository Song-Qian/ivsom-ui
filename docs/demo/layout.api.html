基础用法

常规布局

  • Layout 组件在使用之前，需要有相关的依赖库进行支持，如需阅读：快速开始
  • Layout Row自动响应式父元素的宽度， Column最大的列值为： 30, 溢出则自动换行。
  • Layout Column 默认使用inline-block进行布局，在非Flex类型布局下，请使用HTML 标记语言的【><】首尾接龙法，消除空白折叠导致元素换行。

流式布局

常规布局

  • Layout 组件在使用之前，需要有相关的依赖库进行支持，如需阅读：快速开始
  • Layout Row自动响应式父元素的宽度， Column最大的列值为： 30, 溢出则自动换行。
  • Layout Column 流式布局使用Flex，Flex类型布局下，HTML没有空白折叠导致换行问题，所以无需使用HTML 标记语言的【><】首尾接龙法。

对齐方式

水平对齐

  • Layout 区分水平对齐和垂直对齐方式
  • Layout justify flex 布局下的水平排列方式, 对齐方式可输入值： 'start' | 'end' | 'center' | 'space-around' | 'space-between'
  • Layout align flex 布局下的垂直排列方式, 对齐方式可输入值： 'top' | 'middle' | 'bottom'
 
<i-vsom-row justify="center" flex>
    <i-vsom-col :span="10"><div style='height: 32px; background: #99a9bf; border-radius: 5px;'></div></i-vsom-col>
