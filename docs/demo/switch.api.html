switch Api

  • CDN
  • NodeJS

基础用法

常规switch

  • switch 组件在使用之前，需要有相关的依赖库进行支持，如需阅读：快速开始
<i-vsom-switch v-model="status" ></i-vsom-switch>   
                            

                            var app = new Vue({
                                el : '#app',
                                data : function() {
                                    return {
                                        status: true,
                                    }
                                },
                               
                            })

选择背景用法

  • open-color close-color 分别支持打开关闭switch时背景色的设置
<i-vsom-switch v-model="status" :open-color="background1" :close-color="background2" ></i-vsom-switch>   
                            

                            var app = new Vue({
                                el : '#app',
                                data : function() {
                                    return {
                                        status: true,
                                        background1: "#13CE66",
                                        background2:"#ff4949",
                                    }
                                },
                               
                            })

宽高属性设置

  • width height 支持设置自定义宽高
<i-vsom-switch v-model="status" width="100" height="50" ></i-vsom-switch>   
                            

                            var app = new Vue({
                                el : '#app',
                                data : function() {
                                    return {
                                        status: true,
                                    }
                                },
                               
                            })

禁用用法

  • disabled 支持设置禁用
<i-vsom-switch v-model="status"  ></i-vsom-switch>   
                            

                            var app = new Vue({
                                el : '#app',
                                data : function() {
                                    return {
                                        status: true,
                                    }
                                },
                               
                            })

触发事件

  • 支持change触发事件
<i-vsom-switch v-model="status" @change="change"  ></i-vsom-switch>   
                            

                            var app = new Vue({
                                el : '#app',
                                data : function() {
                                    return {
                                        status: true,
                                    }
                                },
                                methods:{
                                    change(value){
                                       console.log(value)
                                    }
                                 }
                               
                            })
