基础用法

常规计数器

  • inputNumber 组件在使用之前，需要有相关的依赖库进行支持，如需阅读：快速开始
  • v-model双向绑定value值
  • max min 分别表示限制最小最大的值
<i-vsom-input-number v-model="val" :min="1" :max="10" @change="change" ></i-vsom-button>   
                            

                            var app = new Vue({
                                el : '#app',
                                data : function() {
                                    return {
                                        val:5
                                    }
                                },
                                methods:{
                                    change(value){
                                        console.log(value)
                                    }
                                }
                               
                            })
